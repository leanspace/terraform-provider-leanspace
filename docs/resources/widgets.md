# leanspace_widgets (Resource)



## Example Usage

```terraform

  name        = "Terraform Line Widget"
  description = "A line widget created with Terraform"
  type        = "LINE"
  granularity = "second"
  series {
    id          = var.numeric_metric_id
    datasource  = "metric"
    aggregation = "avg"

      operator  = "gt"

}
```
